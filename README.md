# Mini RAG Project

Projeto simples de IA em Python usando RAG (Retrieval-Augmented Generation).

## Estrutura

- **backend**: código Python da API/motor de IA.
- **backend/documents**: documentos de exemplo para teste do RAG.
- **frontend**: pasta para futura interface (web/app).
- **requirements.txt**: dependências Python.

## Como começar

```bash
cd "mini-rag-project"
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python backend/main.py
```
